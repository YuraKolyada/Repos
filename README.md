# RUN DEVELOPMENT VERSION
Prerequisites:
1. Node.js >= v8
2. npm     >= v6 

```javascript
1. npm install
2. cp etc/config.js.sample public/static/config.js
3. npm start
4. open http://localhost:3000 in browser
```

# BUILD PRODUCTION VERSION
Prerequisites:
1. Node.js >= v8
2. npm     >= v6 

```javascript
1. npm install
2. cp etc/config.js.sample public/static/config.js
3. npm run build
```
